# Recun Marketing 18 — Yogii Kumar

Cinematic personal brand site built with Next.js 14, Lenis smooth scroll, GSAP ScrollTrigger, Framer Motion, and React Three Fiber particle field.

## Quick start

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Assets

Bundled from `yodi new/` into:

- `public/audio/ambient-loop.mp3`
- `public/fonts/CormorantGaramond.woff2`, `Satoshi-Variable.woff2`
- `public/images/yogii-portrait.jpg`, `og-image.jpg`
- `public/images/clients/` — vision11, starbucks, rapido
- `app/icon.jpg` — YK monogram (favicon via Next.js App Router)

Service images: `public/dragon.webp`, `public/jupiter.webp`

## Stack

- Next.js 14 App Router
- Tailwind CSS 3
- `@studio-freight/lenis`, GSAP + ScrollTrigger
- Framer Motion 11
- React Three Fiber + custom GLSL particles
